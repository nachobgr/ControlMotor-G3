## Barrido Rítmico (Scanning)
Sistema de selección por pulsador único para pacientes con movilidad mínima.

## Arrastre Sostenido (Drag & Drop)
Mover objetos de un punto a otro sin soltarlos. Mide la fatiga muscular y el control de la presión.  

## Ganancia Adaptativa
Un módulo que ajusta la sensibilidad del mouse para pacientes con rangos de movimiento (ROM) muy limitados.
